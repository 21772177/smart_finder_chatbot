# Quick Start - Mobile Guide

## 🚀 3-Step Quick Start

### Step 1: Open on Mobile
Open your mobile browser and go to:
```
https://buildkit-1695f.web.app
```

### Step 2: Allow Location (Optional)
When prompted, click "Allow" for better results

### Step 3: Start Chatting!
Type your query and hit Send

# 🧭 How to Navigate to OAuth Consent Screen

## 📍 Current Location

You're currently on: **OAuth Overview** page (Google Auth Platform)

**What you see:**
- Left sidebar with: Overview, Branding, Audience, Clients, etc.
- Metrics showing "No data available"
- OAuth token grant rate section

---

## 🎯 Where to Add App Name & Details

**The app name and other details are NOT on this page.**

You need to go to: **OAuth Consent Screen** (different location)

---

## 🚀 Method 1: Direct Link (Easiest)

**Click this link:**
```
https://console.cloud.google.com/apis/credentials/consent?project=buildkit-1695f
```

This will take you directly to the OAuth consent screen where you can add:
- App name
- User support email
- Application home page
- Privacy policy link
- Authorized domains
- Developer contact

---

## 🚀 Method 2: Navigate from Current Page

### Option A: Use Top Navigation

1. **Click the hamburger menu** (☰) in top left
2. **Look for "APIs & Services"** in the menu
3. **Click "APIs & Services"**
4. **Click "OAuth consent screen"** in the left sidebar

### Option B: Use Search

1. **Click the search bar** at the top (says "Search (/) for resources...")
2. **Type:** `OAuth consent screen`
3. **Click on "OAuth consent screen"** from results
4. **Select project:** `buildkit-1695f` if prompted

### Option C: Go to APIs & Services Directly

1. **In the top navigation**, look for project selector
2. **Click on project name** or search for "APIs & Services"
3. **Navigate to:** APIs & Services → OAuth consent screen

# Smart Finder AI Chatbot - Firebase Deployment Package

This package contains a minimal, deployment-ready starter for the Smart Finder AI Chatbot using Firebase Hosting + Firebase Functions (Node.js).  
It includes:
- Frontend (static web chat UI)
- Backend (Firebase Functions) with sample endpoints:
  - /api/query
  - /api/nearby
  - /api/recall
- WhatsApp integration hooks (Meta WhatsApp Cloud API / Twilio notes)
- Environment file examples and deployment instructions

**IMPORTANT:** This is a starter scaffold. You must add your own API keys (OpenAI, Google Maps, WhatsApp/Twilio) and configure OAuth for Google Timeline when enabling recall.
